# Portfolio

"A personal portfolio showcasing my projects, skills, and achievements. Built to highlight coding expertise, creative problem-solving, and continuous learning."

I will be learning "how to use github effectively" as well as "I will learn the concepts for web development and implement that in my portfolio".

## Versions

### HTML/CSS/JS Version
- `index.html`: Registration page with form validation.
- `home.html`: Portfolio home page with embedded CSS and animations.

Open `index.html` in a browser to start.

### React Version
- `portfolio.jsx`: React component for the portfolio.
- `portfolio.css`: Styles for the React version.

To use the React version, set up a React app and import the component.

## Features
- Responsive design
- Space-themed background with animations
- Smooth scrolling navigation
- Project links to GitHub repos
